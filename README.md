# Portfolio Website

A modern and responsive portfolio website built using HTML, CSS, and JavaScript.

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Font Awesome
* Google Fonts
* AOS (Animate On Scroll)

## Features

* Responsive Design
* Dark / Light Mode
* Smooth Scrolling Navigation
* Animated Hero Section
* Tech Stack Showcase
* Project Showcase
* Certifications Section
* Resume Download
* Contact Form
* Mobile-Friendly Navigation

## Project Structure

```text
portfolio/
│
├── assets/
├── index.html
├── style.css
├── script.js
└── README.md
```

## Getting Started

Clone the repository:

```bash
git clone https://github.com/sowmya-sruthidevi/portfolio-1.git
```

Open the project:

```bash
cd portfolio-1
```

Run the website by opening:

```bash
index.html
```
